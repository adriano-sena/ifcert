
## Sobre o IFCERT 

O IFCERT é um projeto que pretende ser a plataforma oficial de gerenciamento de eventos do IFBA-Campus Eunápolis. Abrangendo a criação de eventos, emissão de certificados, submissão de trabalhos e mais funcionalidades futuras. 


## Stack

A stack principal conta com: 

-Laravel 6;
-Mysql 8;
-Bootstrap 4;
